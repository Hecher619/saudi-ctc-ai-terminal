# Contributing to CTC AI Terminal

We welcome high-precision contributions. To ensure system integrity, please follow these "Sniper" standards:

### 🛠️ Pull Request Process
1. **Branching:** Use `feat/` or `fix/` prefixes.
2. **Testing:** All PRs must pass the `npm test` suite before review.
3. **Architecture:** New features must be documented in `docs/ARCHITECTURE.md`.
4. **Security:** Do not include hardcoded credentials.

### 🧪 Code Style
* Use ES6+ syntax.
* Follow the Deno/Edge runtime standards.
* Implement error handling for all external API calls.
